# Heartbeat Tasks

## 每日天气推送（早上 8:00）

**目标：** 每天早上 8:00 推送南京市浦口区天气给孟先生

**执行条件：**
- 当前时间在 08:00 - 08:30 之间
- 飞书对话可访问

**执行步骤：**

1. **获取当前天气数据**
```bash
curl -s "https://wttr.in/Nanjing,Pukou?format=j1"
```

2. **解析并发送消息**
- 提取当前温度、天气状况、湿度、风速
- 提取今日最高/最低温度
- 发送格式化的天气报告到飞书

**消息格式示例：**
```
🌤️ 早上好，孟先生！

📍 南京市浦口区天气
━━━━━━━━━━━━━━━━
🌡️ 温度：XX°C（体感 XX°C）
☀️ 天气：晴朗
💧 湿度：XX%
🌬️ 风速：XX km/h

📅 今日预报
━━━━━━━━━━━━━━━━
最高：XX°C | 最低：XX°C

祝您今天愉快！
```

**记录更新：**
- 每次推送后更新 memory/YYYY-MM-DD.md
- 记录推送时间和天气概况
